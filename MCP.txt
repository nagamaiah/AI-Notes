-- GitHub Codespaces is a cloud-based development environment hosted by GitHub that lets you code directly in the cloud using 
   Visual Studio Code (VS Code) or a browser. 


-- GitHub Codespaces provides:
A complete dev environment in the cloud (Linux-based, Docker-backed).
Pre-configured with tools, extensions, and dependencies.
Integrated with VS Code and the GitHub platform.
Launchable directly from a GitHub repository.
Customizable environments using a devcontainer.json file.

-- Create a Codespace
Go to any GitHub repo (e.g., your own project).
Click the <> Code button.
Select the "Codespaces" tab.
Click "Create codespace on main" (or another branch).
